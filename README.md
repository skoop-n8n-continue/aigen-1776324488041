# Skoop Live Dashboard (v2.0.0)

A modern, "real live" dashboard app optimized for digital signage.

## Features
- **Live Weather**: Real-time weather data from Open-Meteo.
- **Live Markets**: Real-time cryptocurrency prices (BTC/ETH) from Binance.
- **Live News**: Dynamic headlines from HackerNews.
- **Smart Signage**: Auto-scrolling content ensures visibility in non-interactive environments.
- **Auto-Refresh**: Data automatically refreshes every 5 minutes.
- **Offline Capable**: Integrated localStorage fallbacks and PWA support.

## Technical Details
- **Architecture**: Single Page Application (SPA) using vanilla HTML5, CSS3, and JavaScript.
- **Styling**: Modern dark theme using CSS variables and flexbox/grid.
- **Performance**: Optimized with cache-busting asset versioning and `cache: 'no-store'` for API calls.
